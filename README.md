# Defect-detection
基于QT的缺陷检测系统，包括图像检测以及目标检测两个部分。
图像检测包括二值化和边缘检测以及图像矩运算，目标检测使用ONNXRuntime推理yolov5s训练模型，支持GPU加速。
支持奥比中光工业相机检测、视频检测以及图片检测。
